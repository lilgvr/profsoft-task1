### Made using webpack 5, sass

#### NPM Scripts:

> * **dev** runs webpack in development mode
> * **build** runs webpack in production mode
> * **start** runs webpack-dev-server

### Installing dependencies

```bash
npm i
```

### Running the application

```bash
npm start
```
